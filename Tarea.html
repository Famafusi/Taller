Selectores

Un selector es una regla de CSS utilizada para seleccionar elementos HTML y aplicarles estilos. A continuación, se presentan diferentes tipos de selectores:

1. Descendencia

Un selector de descendencia aplica estilos a un elemento solo si está dentro de otro. Ejemplo:

Este es un párrafo hijo dentro de un div.

2. Selectores Básicos

  • Selector de etiqueta: aplica estilos a todas las instancias de una etiqueta específica. Ejemplo: el selector `p` aplica estilos a todos los párrafos.
  • Selector de clase: aplica estilos a los elementos con una clase específica. Ejemplo: `.example` aplica estilos a los elementos con esa clase.
  • Selector de id: aplica estilos a un elemento con un id específico. Ejemplo: `#unique` aplica estilos a un elemento con id="unique".

3. Selectores de atributo

Los selectores de atributo permiten seleccionar elementos en función de sus atributos. Ejemplo:

Enlace con target="_blank"

4. Selectores Combinados

Los selectores combinados permiten seleccionar elementos en función de múltiples condiciones. Ejemplo: el selector `div p` selecciona todos los párrafos dentro de un div.
